# 前言

基于Vue2的头条Web APP

## 技术栈

Vue + Vue-router + Vuex

## 项目运行
```
git clone https://github.com/skykingit/vue-toutiao.git
cd vue-toutiao
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
